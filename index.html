The Mesh

We've taken the data point provided by NASA and decided to create a 3D mesh out of it utilizing Python and its many libraries.

Textures

There was a lot of other data that we also had to visualize like walkable paths and the slope map. We created textures out of this data and applied it to our mesh.

The App

We're going to make this all come together with our app. We are using a tool called Unity to help us with this process and make a friendly user interface.
